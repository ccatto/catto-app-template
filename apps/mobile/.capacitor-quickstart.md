# Capacitor Quick Reference

## Daily Development

### Web Development (Most of the time)

```bash
# From root
yarn dev:frontend
yarn dev:backend
```

### Mobile Testing (When needed)

```bash
# Build and test on iOS
cd apps/mobile
yarn build
yarn ios:open
```

## Common Commands

```bash
# Build mobile app
yarn build                    # Full build: frontend → copy → sync

# iOS
yarn ios:open                 # Open Xcode
yarn ios:run                  # Run on simulator

# Sync only (if you manually built frontend)
yarn sync                     # Sync www → native projects
```

## Build Output Locations

- **Frontend build**: `apps/frontend/out/`
- **Mobile www**: `apps/mobile/www/` (copied from frontend/out)
- **iOS project**: `apps/mobile/ios/`

## Debugging

### iOS Simulator

1. Run app in simulator
2. Safari → Develop → Simulator → MyApp
3. Open Web Inspector to see console

### Xcode

1. View → Navigators → Show Report Navigator (⌘9)
2. Click build/run to see logs
3. Debug area (⇧⌘Y) shows runtime logs

## Next.js Configs

- **Web**: `apps/frontend/next.config.ts` (server mode)
- **Mobile**: `apps/frontend/next.config.mobile.ts` (static export)

Mobile build automatically uses the mobile config.
